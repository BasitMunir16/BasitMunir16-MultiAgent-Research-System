# 🧠 Research Prospect AI Agent

An AI-powered agent built using RelevanceAI that converts LinkedIn profiles into structured, sales-ready insights.

---

## 👨‍💻 Author
Basit Munir

---

## 🚀 Overview

This agent takes a LinkedIn URL as input, extracts profile data, and uses a Large Language Model (Gemini AI) to generate a concise 300-word professional summary.

---

## ⚙️ How It Works

1. Input: LinkedIn profile URL  
2. Data Extraction: Scrapes profile using RelevanceAI tool  
3. AI Processing: Gemini LLM generates structured summary  
4. Output: Sales-ready prospect/company overview  

---

## 📤 Output Includes

- Company/Person overview  
- Products & services  
- Mission & values  
- Location details  
- Key sales talking points  

---

## 🎯 Use Case

Designed for sales teams, recruiters, and business analysts to quickly understand prospects before meetings.

---

## 🛠️ Tech Stack

- RelevanceAI  
- LinkedIn Data Tool  
- Google Gemini AI
